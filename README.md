
# Reference

This is a new hugo based static site for thetechnohack blog with netlify-cms backed-in. The site is based on hugo-terminal theme by panr.

# Folder structure

```
├───archetypes
├───assets
│   ├───css
│   │   └───color
│   ├───fonts
│   └───js
├───content
│   └───posts
├───layouts
│   ├───partials
│   ├───shortcodes
│   └───_default
└───static
    ├───admin
    ├───assets
    ├───images
    └───img
        └───favicon
```
